# AirbusStyleCase
# 空客（风味）铝型材机箱
为自己的显卡坞做一个铝型材机箱和一个空客风味的面板，包含一个空客ECAM风格的小屏幕用来监控电脑传感器信息，旋钮控制显卡坞散热风扇的转速。  
视频介绍地址：https://www.bilibili.com/video/BV1AxmsBbEEY/  
面板3f文件开源到拓竹MakerWorld：https://makerworld.com.cn/zh/models/1942075-kong-ke-feng-wei-xian-qia-wu-mian-ban#profileId-2157028
